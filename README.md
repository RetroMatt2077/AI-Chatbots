# 🤖 AI Chatbots Collection

A collection of Python-based AI chatbots ranging from simple rule-based assistants to advanced LLM-powered conversational agents with GUI.

![OpenAI](https://img.shields.io/badge/OpenRouter-000000?style=for-the-badge&logo=openai&logoColor=white)

## 📋 Projects Included

| File                        | Type                  | Description |
|----------------------------|-----------------------|-----------|
| `aichatbot1.py`            | Console               | Simple rule-based chatbot with multiple personalities (friendly, sarcastic, professional) and memory. |
| `aichatbot2.py`            | GUI (Kivy)            | Basic keyword-driven chatbot with persistent memory. |
| `jarvis-ai-bot.py`         | Console + OpenAI      | Lightweight Jarvis-style assistant using OpenRouter / OpenAI API. |
| `jarvis-chatbot-ai.py`     | GUI (Kivy)            | Advanced Jarvis assistant with mood system, idle messages, topic tracking, and LLM integration. |
| `RPGChatbot.py`            | GUI (Kivy)            | Full-featured AI chatbot with multiple personalities including a full **Dungeon Master RPG mode**, notes system, and conversation history. |

## ✨ Features

- **Rule-based chatbots** – No API key needed
- **LLM-powered bots** – Using OpenRouter (Llama 3.1, etc.)
- **Beautiful Kivy GUIs** – Clean chat interfaces
- **Personalities & Modes** – Jarvis, Dungeon Master, Teacher, Comedian, etc.
- **Memory & Persistence** – Saves notes, chat history, and user interests
- **RPG Mode** – Play text-based adventures with a smart Dungeon Master
